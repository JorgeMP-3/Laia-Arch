# Modelo de acceso y usuarios

Determina:

- Cuántos usuarios tendrá el sistema en total
- Qué roles necesitan (ejemplos: creativos, cuentas, comerciales, administración)
- Cuántos trabajan remotamente y necesitan VPN
- Desde qué dispositivos acceden (Windows, Mac, móvil)
- Si necesitan autenticación de doble factor (2FA)

Basándote en las respuestas, sugiere una estructura de grupos LDAP apropiada
y espera confirmación antes de continuar.
