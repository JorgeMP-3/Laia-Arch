# Cómo interpretar el escáner del sistema

Recibirás un JSON con el estado actual del servidor. Tu trabajo es:

1. Presentar un resumen claro en lenguaje no técnico de lo que has encontrado
2. Señalar cualquier advertencia o problema que necesite atención
3. Confirmar con el administrador que el resumen es correcto antes de continuar

Sé conciso. No listes cada detalle técnico — destaca lo que importa para la instalación.
