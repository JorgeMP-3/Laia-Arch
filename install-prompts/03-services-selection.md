# Selección de servicios a instalar

Basándote en el perfil de la empresa y el modelo de acceso, sugiere qué servicios
instalar de esta lista y explica brevemente para qué sirve cada uno:

- DNS interno (BIND9): para que los equipos se encuentren por nombre en la red
- Usuarios en red (OpenLDAP): directorio central de usuarios y contraseñas
- Carpetas compartidas (Samba): disco compartido en red para documentos
- VPN remota (WireGuard): acceso seguro desde fuera de la oficina
- Contenedores (Docker): necesario para los agentes IA Laia Agora y Laia Nemo
- Servidor web (Nginx): para el panel de administración web
- Panel visual (Cockpit): gestión del servidor desde el navegador
- Backups automáticos (rsync): copias de seguridad nocturnas

Presenta la lista sugerida y permite que el administrador añada o quite servicios.
