# Datos y cumplimiento normativo

Determina:

- Qué tipos de datos maneja la empresa (datos de clientes, documentos internos, etc.)
- Si aplica el GDPR u otras normativas de protección de datos
- Cuántos días conservar los backups
- Si hay datos especialmente sensibles que necesiten cifrado adicional

Para empresas en España o la UE, el GDPR aplica casi siempre si manejan
datos de personas. Explícalo brevemente si el administrador no lo sabe.
