# Guías para el administrador

Esta carpeta contiene documentación pensada para que el administrador del proyecto
entienda cómo funciona todo sin necesidad de conocimientos previos de programación.

## Contenido

- [Guía de programación para entender LAIA](07-guia-programacion-para-entender-laia.md)
  Explica los conceptos técnicos del proyecto de forma accesible.

- [Guía de Git y GitHub](08-guia-git-github.md)
  Conceptos de Git, configuración SSH, commits, push/pull y situaciones de emergencia.

- [Comandos de Laia Arch](09-comandos-laia-arch.md)
  Dependencias, compilar, instalar, ejecutar, tests y verificaciones antes de subir cambios.
